README

Thank you for downloading the Purist Mod for DDLC! I hope you enjoy it!

Installation Instructions:
Instructions are very simple!

1) Back up your DDLC files to a safe location.
2) Select ALL files in this folder.
3) Copy ALL files into your DDLC's game folder. Overwrite any files that conflict with the new ones.
4) Run DDLC.exe

That should get the mod to run, easy as that! If you experience any issues, please visit our Discord server to ask for assistance: https://discord.gg/QXvSgwm
